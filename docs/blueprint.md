# **App Name**: OpenMind OS

## Core Features:

- User Authentication: Secure user registration, login via Google or email/password, and profile management with protected routes.
- Goal Management System: Users can create, track, categorize, and manage personal goals, setting deadlines and monitoring progress with intuitive controls.
- Task Management System: Detailed task creation including estimated/actual time, completion toggle, deadlines, tagging, and filtering capabilities.
- Interactive Knowledge Graph: A dynamic, D3.js-powered visual graph view for knowledge entries, allowing users to define and manually link skills, topics, goals, and notes.
- Personalized Analytics Dashboard: An intuitive dashboard displaying core productivity metrics, task completion progress, activity trends, and mocked skill radar charts using Recharts.
- AI-Powered Learning Roadmap Tool: A tool that generates personalized learning roadmaps, including weekly breakdowns and daily task suggestions, based on target roles and desired timelines.
- Cognitive Insights Display: A section providing key cognitive metrics like consistency scores, burnout risk indicators, and learning velocity calculated from user activity.

## Style Guidelines:

- The primary accent color is a deep, intellectual blue (#335E99), signifying clarity and focus for a cognitive system. The background is a very light, desaturated blue (#EEF1F5) to create an open and minimal canvas. An accent color of vibrant cyan (#3DBBDD) is used for interactive elements and highlights, bringing an energetic, innovative touch.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-savvy feel. Body text font: 'Inter' (sans-serif) for neutral, clear, and highly readable content, ensuring accessibility and modularity across the OS.
- Utilize a clean set of minimalist, line-art vector icons that complement the modern UI and maintain visual consistency throughout the application.
- A responsive, clean, and modular layout with consistent use of rounded cards and soft shadows for clear content segmentation and visual hierarchy. Supports both light and dark modes.
- Smooth and subtle UI transitions and micro-interactions, implemented with Framer Motion, to enhance user feedback and provide an elegant, fluid experience during navigation and data updates. Includes loading states, skeleton loaders, and empty states.
